# Deadline Guardian

AI-powered productivity companion that helps students and professionals avoid missing deadlines.

## Features

- AI Deadline Risk Prediction
- Smart Task Prioritization
- AI Planner
- Calendar
- Analytics Dashboard
- AI Chat Assistant
- Pomodoro Timer
- JWT Authentication

## Tech Stack

Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

Backend
- Express.js
- PostgreSQL
- Drizzle ORM

AI
- Gemini API
